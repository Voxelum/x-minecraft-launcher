```bash
brew tap voxelum/xmcl
brew install --cask --no-quarantine voxelum/xmcl
```

## Features


- üì• **Download & auto complete**. Support download `Minecraft`, `Forge`, `Fabric`, `Quilt`, `OptiFine`, `JVM` from official or third party mirrors.
- ‚ö°Ô∏è **Download Fast**. Reuse socket via HTTP/HTTPS agents, and download the files in parts concurrently.
- üíª **Cross platform**. The launcher is based on Electron, and supports üóî Windows 10/11, üçé MacOS, and üêß Linux.
- üìö **Multi-Instancing**. Users can create multiple instances to isolate the different versions, mods and launch settings.
- üóÇ **Manage all resources**. Use (hard/symbolic) links to install resources in instances, keep your disk usage optimal. No copies of mods everywhere! üòÜ
- üî• **Built-in support of CurseForge, Modrinth**. You can download resources inside the launcher.
- üì¶ **Support import/export** CurseForge & Modrinth modpacks with compliance!
- üîí **Support multiple account systems**. Built-in Microsoft login and Mojang Yggdrasil API. It also has builtin support of [ely.by](https://ely.by/) and [littleskin.cn](https://littleskin.cn). You can also add third-party authentication servers!
- üîó **Peer to peer connection between users**. You can play multiplayer over LAN even you are not in same physical LAN!
- üîë **Code sign & modern packaging**. Under Windows, you can use `appx` and `appinstaller` to install the app. You won't receive blocking messages from your browser or see SmartScreen errors anymore! üòé
