## 概要
本次改动对项目中的简体中文本地化进行了“激进式”风格统一与简化，目标为：
- 面向玩家但工具化（短句、直接、少修饰）
- 统一语气（由较正式/敬语向更简洁、口语化但准确的描述）
- 移除冗余语气词（如过多的“请”、“！”）并压缩超长说明
- 修正少量错字/拼写/重复段落
- 清理遗留的备份文件（.bak）

> 说明：本次改动以 `zh-CN` 翻译为主（主要在 `xmcl-keystone-ui`），并同步调整了 `xmcl-electron-app` 的 zh-CN 文本以保持一致性。

## 修改文件清单
- xmcl-keystone-ui/locales/zh-CN.yaml
- xmcl-electron-app/main/locales/zh-CN.yaml

（同时删除了工作区中两个备份文件：`xmcl-keystone-ui/locales/zh-CN.yaml.bak`、`xmcl-keystone-ui/locales/zh-CN.yaml.bak2`）

## 主要改动点（按模块）
- 全局：把原文中大量"您"替换为"你"（先前已完成），去除感叹号并缩短冗长句子。
- App / 实例：简化创建、分享对话的描述；对话更短、强调动作（例如“选择一项导入”替代长句）。
- 登录 / 认证：将“通证 / 令牌”相关错误提示缩短为更直接的报错信息（例如“获取 Microsoft 令牌失败，重试或检查网络”），并规范了用词（统一使用“令牌”/“账户”）。
- 启动 & 启动失败（launch、launchBlocked、launchFailed）：把复杂的诊断或建议压缩为简短动作项（如“检查 Java 或提升权限”），保留关键占位符和 HTML 标签。
- 诊断（diagnosis）：将几乎所有长句替换为精简说明（例如"启动器会自动下载缺失资源"等），同时保持原有逻辑（自动下载/重新安装等）。
- 设置（setting）：重写/压缩多处设定说明（主题、背景、GPU、代理、BMCLAPI说明等）以更清晰、短句化呈现。
- 反馈（feedback）与教程（tutorial）：调整介绍、提示文字为更直接的通知或操作句。
- P2P / 联机（multiplayer）：压缩令牌说明、ICE 收集解释，减少 HTML 垃圾与重复描述。
- 其他：模组、实例、资源、导入等相关字符串的精简与一致性修正。
